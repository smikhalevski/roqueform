Interface Field<Value, Mixin>

The field that holds a value and provides means to update it. Fields can be enhanced by plugins that provide such things as integration with rendering and validation libraries.

Type Parameters

  • Value = any

    The root field value.

  • Mixin = unknown

    The mixin added by the plugin.

Properties

Readonly isTransient

isTransient: boolean

true if the value was last updated using setTransientValue, or false otherwise.

Readonly key

key: any

The key in the parent value that corresponds to the value controlled by the field, or null if there's no parent.

Readonly parent

parent: null | Field<any, Mixin> & Mixin

The parent field from which this one was derived, or null if there's no parent.

Readonly value

value: Value

The current value of the field.
